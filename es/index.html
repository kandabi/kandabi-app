Proyectos

Juego de drones Tello

Desarrollé un video juego de drones con realidad aumentada, donde el jugador es capaz de volar un dron real y úsalo para disparar armas virtuales.

El juego se creó utilizando el motor de juego Unity para dispositivos móviles.

Para mostrar las imágenes del dron se desarrolló un complemento de renderizado nativo para decodificar la transmisión de video.

Fue desarrollado como desarrollador autónomo.

OpenFTP

Desarrollé un software de código abierto para transferir archivos grandes entre computadoras a través de Internet de forma segura y rápida.

Escrito en C++, usando el marco Qt para UI, junto a OpenSSL para un cifrado de red seguro.

Desarrollé como un proyecto de código abierto, alojado en Github.

Aplicación de comparación de mapas

Desarrollé un sitio web de comparación de mapas en paralelo destinado a mostrar y analizar archivos de mapas grandes exportados utilizando imágenes aéreas.

Capaz de dibujar: formas, polígonos, marcadores y líneas con medidas de área automáticas.

Escrito en Javascript y jQuery, utilizando el Leaflet biblioteca de mapas y alojado en Netlify.

Fue desarrollado como desarrollador autónomo.

Detectores de objetos

Desarrollé detectores de objetos basados en aprendizaje automático, capaz de detectar personas, vehículos, animales y muchos tipos diferentes de objetos complejos.

Escrito en C++ y Python, entrenado usando el marco YOLOv4, junto con OpenCV biblioteca para visión artificial.

Fue desarrollado como desarrollador autónomo.

Simulación de física de píxeles

Desarrolló una atractiva simulación de física de partículas, simulando cómo el agua y la arena interactúan entre sí.

Escrito en C++, usando Pixel Game Engine de olc.

Desarrollado como un proyecto de hobby, lanzado como código abierto en Github.
